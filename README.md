Тестовое задание.

	Нужно написать консольный PHP-скрипт, который работает с API M4 по приложенной документации.
	Который:
	1. Выполнить авторизацию через API.
	2. Получить URL сервиса SD из ответа авторизации.
	3. Получить список заявок, изменённых за последние 3 дня.
	4. Если список заявок пустой или содержит меньше двух элементов, скрипт не должен падать.
	Он должен вывести сообщение:
	"Недостаточно заявок для выполнения тестового сценария"
	и завершиться с кодом 0.
	5. Если заявок две или больше:
	- взять вторую заявку из списка;
	- получить по ней детальную информацию;
	- вывести в консоль: taskId, req, caption, status/statusName.
	6. Загрузить на сервер два любых изображения.
	7. Прикрепить эти два изображения к выбранной заявке.
	8. Добавить к заявке публичный комментарий:
	"Тестовый комментарий от кандидата: {ФИО}, {дата-время}"
	9. Выполнить logout.

	Технические требования:

	1. PHP 7.4+ или PHP 8+.
	2. Можно использовать Composer и HTTP-клиент, например Guzzle.
	3. Нельзя хранить логин/пароль/token прямо в коде.
	Доступы должны передаваться через .env, config.php или параметры запуска.
